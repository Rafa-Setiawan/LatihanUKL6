Program di atas digunakan untuk menjumlahkan dua buah matriks (A dan B) berukuran sama.\

Pengguna memasukkan jumlah baris dan kolom matriks.

Program membuat tiga array 2D:

matriksA → menyimpan elemen matriks A

matriksB → menyimpan elemen matriks B

hasil → menyimpan hasil penjumlahan

Pengguna mengisi nilai untuk setiap elemen matriks A dan B.Menginput dua matriks (A dan B), lalu menghitung dan menampilkan hasil penjumlahan A + B.
